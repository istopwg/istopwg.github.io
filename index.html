ISTO-PWG Github Projects

The ISTO Printer Working Group maintains several projects on Github for the active workgroups and registries it maintains.

ISTO-PWG Web Site

Projects

ippsample

The ippsample project provides sample implementations of an IPP Client, Proxy, Server, and other tools. It is based on the CUPS source code and is maintained by the PWG IPP workgroup.

Project Page

ippeveselfcert

The ippeveselfcert project contains the IPP Everywhere® self-certification tools.

Project Page

ippregistry

The ippregistry project is used to maintain the IANA IPP Registry as well as the Pending IPP Registry on the PWG IPP workgroup home page.

Project Page

pwg-semantic-model

The pwg-semantic-model project is used to maintain the PWG Semantic Model XML schema.
